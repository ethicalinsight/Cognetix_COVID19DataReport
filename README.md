# Cognetix_COVID19DataReport
# COVID-19 Data Report

## 📌 Objective
Analyze global/country-level COVID-19 data and visualize case progression.

## 📊 Dataset
- Source: [COVID-19 Report](https://www.kaggle.com/datasets/imdevskp/corona-virus-report)

## 🛠️ Tools & Libraries
- Python, Pandas, Matplotlib, Seaborn, Plotly (optional)

## 🚀 Steps
1. Load and clean dataset (date formatting, missing values).
2. Filter by country.
3. Plot daily confirmed cases, deaths, recoveries.
4. Add rolling averages and trend lines.
5. Compare multiple countries.

## 📈 Deliverables
- Google Colab Notebook
- Trend charts
- Country comparison visuals
